1 Controller メソッド一覧

ReservationController 必須
予約管理コントローラーの全メソッド
メソッド名 HTTP URL 引数 戻り値 処理概要
list GET /reservations Model model String 予約一覧を取得してModelに追加し、予約管理画面を返却
create POST /reservations @Valid ReservationForm form, BindingResult result String 入力チェック後、予約情報を登録して一覧画面へリダイレクト
update POST /reservations/{id} @PathVariable Long id, @Valid ReservationForm form, BindingResult result String 指定IDの予約情報を更新して一覧画面へリダイレクト
delete POST /reservations/{id}/delete @PathVariable Long id String 指定IDの予約情報を削除して一覧画面へリダイレクト
search GET /reservations/search String keyword, Model model String キーワード条件で予約情報を検索して結果を画面表示する
changeStatus POST /reservations/{id}/status @PathVariable Long id, String status String 指定IDの予約ステータスを変更して一覧画面へリダイレクト

2 Service メソッド一覧

ReservationService 必須
予約サービスの全メソッド
メソッド名 引数 戻り値 処理概要 呼び出すRepository
findAll なし List<Reservation> 全予約情報を取得する reservationRepository.findAll()
findById Long id Reservation ID指定で予約情報を取得する reservationRepository.findById(id)
create ReservationForm form void フォーム入力値をEntityへ変換して予約情報を登録する reservationRepository.save(reservation)
update Long id, ReservationForm form void 既存の予約情報を更新する reservationRepository.update(reservation)
delete Long id void 指定IDの予約情報を削除する reservationRepository.delete(id)
search String keyword List<Reservation> 氏名やメールアドレスで予約情報を検索する reservationRepository.search(keyword)
changeStatus Long id, String status void 予約のステータスを変更する reservationRepository.update(reservation)
InquiryService 任意
お問い合わせサービスの全メソッド
メソッド名 引数 戻り値 処理概要 呼び出すRepository
findAll なし List<Inquiry> 全お問い合わせ情報を取得する inquiryRepository.findAll()
findById Long id Inquiry ID指定でお問い合わせ情報を取得する inquiryRepository.findById(id)
create InquiryForm form void フォーム入力値をEntityへ変換してお問い合わせ情報を登録する inquiryRepository.save(inquiry)
update Long id, InquiryForm form void 既存のお問い合わせ情報を更新する inquiryRepository.update(inquiry)
delete Long id void 指定IDのお問い合わせ情報を削除する inquiryRepository.delete(id)
search String keyword List<Inquiry> 氏名・件名・本文でお問い合わせ情報を検索する inquiryRepository.search(keyword)
changeStatus Long id, String status void お問い合わせの対応ステータスを変更する inquiryRepository.update(inquiry)
