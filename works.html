香りのワークショップ「inori」

作品概要 必須
どんなサービス・サイトか、ターゲットユーザーは誰か

香りのワークショップ「inori」をテーマにした予約サイトを制作した。 浅草で自分だけの香りを作る体験を想定し、体験内容・料金・スケジュール・予約までを分かりやすく伝える構成にしている。日本語と英語の2言語に対応し、外国人観光客にも利用しやすいサイトを目指した。

使用技術 必須

フロントエンド：HTML / CSS / JavaScript

バックエンド：HTML / CSS / JavaScript

データベース：

その他：Git / Figma

担当範囲 必須
企画・デザイン・フロントエンド・バックエンドなど

企画、サイト構成設計、ワイヤーフレーム作成、UIデザイン（Figma）、配色設計、写真選定、日本語・英語ページのデザインを担当

工夫した点・アピールポイント 必須

香りのワークショップの雰囲気が伝わるように、香水制作の様子が分かる写真を大きく配置し、体験イメージを持ちやすいデザインにした。 また、白とピンクベージュを基調とした柔らかい配色でサイト全体を統一し、香りやリラックス感を表現した。 さらに、体験紹介から予約までの流れが分かりやすくなるよう、ページ構成や情報整理を工夫した。
